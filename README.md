Họ tên: Pham Nguyen Hung

Lớp: D18CNPM2

## Hướng dẫn cài đặt
1. Chạy `composer install`
2. Copy `.env.example` thành `.env` và điền DB, API Keys.
3. Chạy `php artisan migrate`
4. Chạy `php artisan serve`

## Cấu hình
- Google Redirect: http://127.0.0.1:8000/auth/google/callback
- Facebook Redirect: http://127.0.0.1:8000/auth/facebook/callback
